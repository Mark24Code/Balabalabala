# 数据类型

----
## 1.数据类型

ES5有5种简单数据类型（也称作基本数据类型）:

* Undefined
* Null
* Boolean
* Number
* String

ES6添加了一个

* Symbol

还有一种复杂的数据类型类型

* Object

---

### Q&A:

1. 这么少的数据类型，够表示所有数据么？

> 由于ES数据类型具有动态性，因此的确没有再定义其他数据类型的必要。——《红宝书(3rd)》P23
